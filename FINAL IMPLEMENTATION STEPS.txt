4. **events** - id, tenant_id, source, raw_json, timestamp, processed_flag

### **Step 13: Create GTM Lens Prompt (15 min)**

Same structure as Theme lens, different prompt

### **Step 15: Create Positioning Lens Prompt (15 min)**

Write prompt for Positioning:

- Same structure
- Different examples

### **Step 16: Build Positioning Lens Function (15 min)**

Same structure as Theme lens, different prompt

### **Step 17: Create Category Gate Logic (15 min)**

Build function that:
- Takes one Insight and runs all three lenses
- Collects three confidence scores
- Finds maximum confidence score
- If max confidence >= 0.7: assigns corresponding category to insight

- Returns updated Insight object

**Critical:** This is where category gets assigned, NOT in the lenses
